
# Product Roadmap
## MappingXML2PipelineJSON

This roadmap outlines the evolution of the MappingXML2PipelineJSON platform from MVP to enterprise-ready migration tooling.

---

# Phase 1 — MVP

Goal: Build a working pipeline conversion prototype.

Core capabilities:

- XML upload
- source ETL detection
- pipeline parsing
- canonical JSON generation
- Airflow pipeline conversion
- basic validation
- JSON export

Supported source tools:

- Informatica PowerCenter

Supported target platforms:

- Apache Airflow

Deliverable:

Working prototype capable of converting simple pipelines.

---

# Phase 2 — Production SaaS (v1)

Goal: Launch a production-ready SaaS platform.

New capabilities:

- multi-project dashboard
- pipeline visualization
- validation reports
- conversion history
- authentication
- subscription billing
- REST API access

Supported platforms expanded:

- Azure Data Factory
- Apache Airflow
- Databricks Workflows
- Dagster

Infrastructure:

- scalable API services
- background job queues
- artifact storage

---

# Phase 3 — Advanced Migration Platform (v2)

Goal: Support enterprise migration workflows.

New capabilities:

- AI-assisted transformation mapping
- automated SQL generation
- pipeline lineage extraction
- migration complexity scoring
- batch conversion tools
- enterprise team collaboration

Enterprise features:

- role-based access control
- audit logs
- bulk migration reports
- custom adapters

---

# Phase 4 — Enterprise Data Platform Migration Suite

Goal: Become a full migration accelerator for enterprise data platforms.

Future capabilities:

- automated ETL modernization assessments
- large-scale migration automation
- pipeline performance optimization
- integration with CI/CD pipelines
- migration monitoring dashboards
