# Demo Assets

Place the following files in this directory:

- `etl-modernization-platform-demo.mp4` — 90-second product demo video
- `etl-modernization-platform-thumbnail.png` — Video thumbnail image

## Recording Guidelines

1. Show the full pipeline: Upload → Detect → Parse → Normalize → Convert
2. Highlight the Conversion Timeline showing step-by-step progress
3. Show multi-platform conversion (e.g., Airflow + ADF)
4. Show the AI Settings panel
5. Keep it under 90 seconds
